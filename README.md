## ğŸŒ¤ï¸ Weather Widget Setup

1. Get an API key from [AccuWeather](https://developer.accuweather.com/)
2. Find your City ID from [AccuWeather](https://www.accuweather.com/)
3. Click the weather widget to configure:
   - Enter your API key
   - Enter your city ID
